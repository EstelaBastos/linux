# Explorando a pasta `/home` no Linux
A pasta `/home` é como a **casa dos usuários** no S.O. 
Cada pessoa que usa o *sistema* tem uma pasta aqui com seus arquivos.

---
## Estrutura de usuários

```bash
/home/ana
/home/joao
/home/pedro
```
